# 🚀 SkillSync — Connect Skills, Build Teams, and Grow Together

SkillSync is a smart collaboration platform that helps students and professionals **discover teammates, sync skills, and work on impactful projects together**. Whether you’re building your first app, contributing to open source, or joining a hackathon — SkillSync helps you find the *perfect match* for your goals.

---

## 🌟 Features

- 🔍 **Skill-Based Matching:** Find collaborators based on your skills, interests, and goals.  
- 💬 **Nearby Connect:** Find and connect with professionals in your vicinity for instant collaboration.  
- 🧠 **Mock Interviews:** Practice technical and behavioral interviews with real-time feedback.
- 🛠️ **Privacy & Security:** Your personal and professional data is secured with enterprise-grade protocols. 
- 📈 **AI-Powered Matching:** Smart algorithms connect you with the right professionals based on skills and goals.

---

## 🛠️ Tools & Technologies

### Frontend
- **Framework & UI:** React, React Router
- **Styling & Design:** Tailwind CSS, Lucide Icons
- **State & Utilities:** Axios, React Hot Toast, Infinite Scroll
- **Maps & Geolocation:** Leaflet, React Leaflet
- **Build & Dev Tools:** Vite, Vite Plugin React
- **Linting & Types:** ESLint, React Hooks ESLint Plugin, TypeScript types for React

### Backend
- **Runtime & Framework:** Node.js, Express
- **Database & ORM:** MongoDB (Mongoose)
- **Authentication & Security:** bcrypt, bcryptjs, jsonwebtoken, crypto-js, dotenv
- **Validation:** Joi, express-validator
- **File Handling:** Multer
- **API & HTTP:** Axios, CORS, Body-Parse
- **Email Services:** Nodemailer
- **Firebase Services:** firebase-admin
- **Dev Tools:** Nodemon

## ⚙️ Setup Instructions

1. **Clone the repository**
     ```bash
     git clone https://github.com/mahos18/SkillSync.git
     cd SkillSync-main

2. **Install Dependancies  Frontend**
   
     ```Frontend
     cd SkilllSync-main 
     npm install
     ```
     
     **Backend**
   
     ```Backend
     cd Backend
     npm install
     
4. **Set up environment variables**
   
    ```backend .env
    MONGO_URI=<your_database_url>
    JWT_SECRET=<your_secret_key>
    PORT=8000
    
6. **Run the development server**
   
    **Frontend**
   
    ```Frontend
    cd SkillSync-main
    npm run dev
    ```
    
     **Backend**
   
    ```Backend
    cd Backend
    npm start
    ```
